Spotify Angular

This is an Angular 4 Application which uses Spotify Web API to search for Music according to Artist, Album or song, also the user can play preview of the track.

Angular CLI

Angular 4 (latest)
  • Check out the Latest angular 4 Docs Angular 4
  • Start with 
    npm install
  • To launch the application use
    ng serve

    • Spotify web API

    Developer API
  • Check out the Web API Docs Artist Spotify API
  • If you want enhance, get your own Client ID, Secret ID.
  • For spotify credentials have an account and get the API Keys.

    • Features

    what is offered
  • You can search for the artist, song, Album.
  • Get the list of search and dive through.
  • You can also preview the song.
  • Else just click on check out in Spotify.

    • ** If preview Doesn't work, The feature is not available for certain album or artist, try another
